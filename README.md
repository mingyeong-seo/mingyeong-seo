# 서민경 | mingyeong-seo

## Tech
- Frontend
  - HTML, CSS를 활용한 기본적인 화면 구성
  - JavaScript를 통한 이벤트 처리 및 데이터 흐름 이해
  - React 기반 컴포넌트 구조 및 상태 관리 학습 중
- Backend: Java, Spring Boot, JPA (기본적인 CRUD 및 API 이해)
- Tools: Git, GitHub
